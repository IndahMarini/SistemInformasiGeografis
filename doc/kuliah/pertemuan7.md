**PEMBAHASAN**

**Pengertian MapServer**

Map server merupakan aplikasi pemetaan GIS yang dioperasikan oleh sistem operasi seperti linux dan windows yang bersifat opensource yang berfungsi untuk menbaca data graphic.

**Fitur – fitur MapServer**

- **--** Menampilkan data spasial format vektor seperti PostGIS dan ESRI dengan library ORG
- **--** Menampilkan data raster dengan format TIFF/GeoTIFF, EPPL7 dengan library GDAL
- **--** Menggunakan quadtree dlm indexing data spasial dan dilakukan dengan cepat
- **--** Dapat dikembangkan dengan tampilan keluaran yang dapat diatur dengan template.
- **--** Dapat melakukan seleksi objek berdasarkan nilai , titik, area atau berdasarkan objek spasial tertentu
- **--** Mendukung endering dengan font TrueType
- **--** Mendukung rendering karakter berbagai font
- **--** Menampilkan peta tematik menggunakan ekspresi logic dan regaular

**Cara install MapServer**

- **--** Jika tidak menggunakan centos maka jalankan melalui virtual box
- **--** Setelah itu pastikan koneksi jaringan virtuan box bisa diakses dari komputer host
- **--** Buka terminal , login sebagai root
- **--** Ketikkan &quot;#install mapserver 5
- **--** Tunggu hingga proses install selesai.

**Pengertian MapProxy**

MapProxy merupakan program yang berfungsi sebagai wadah yang menampung hasil gambar dari MapServer agar bisa direduksi.

**Cara install MapProxy**

- **--** Lalu &quot;#pip install mapproxy&quot;
